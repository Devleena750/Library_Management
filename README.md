# 📚 Library Management System

## 🔹 Technologies Used

* Java (Swing)
* Oracle Database
* JDBC

## 🔹 Features

* Add Books
* View Books
* Manage Members
* Issue Books

## 🔹 How to Run

1. Open SQL*Plus and run `library.sql`
2. Update DBConnection.java with your password
3. Run the project in IntelliJ

## 🔹 Database Connection

```java
jdbc:oracle:thin:@localhost:1521:xe
```

Username: system
Password: your_password
